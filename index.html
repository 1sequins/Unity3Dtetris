Welcome to Unity3D Tetris Documentation Page.

This is a project page for a Tetris Game developed by Unity3D, the purpose of the game is to compare how users brain activities react when users are in a normal mode vs in a panic mode. In this game, we give users 3 mode of game play , dynamic , Dynamic Manual and Constant

For Dynamic Mode - Game Speed increases when users clear a line Fr Dynamic Manual - Game Speed increases when users clear a line , however users can control the speed of the game For Constant - Game Speed do not change when users clear the line,

To play the game demo a Unity Web player can be played here http://bit.ly/11pYNpJ

For those who do not have unity web player please download it here http://unity3d.com/webplayer/
