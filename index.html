Welcome to Unity3D Tetris.

This is a project page for a Tetris Game developed by Unity3D, the purpose of the game is for research and experiment purpose for our laboratory. Our goals in this experiment is to compare how brain activities react when users are in a normal mode vs in a panic mode. In this game, we give users 3 mode of game play Dynamic,Dynamic Manual and Constant

For Dynamic Mode - Game Speed increases when users clear a line Fr Dynamic Manual - Game Speed increases when users clear a line , however users can control the speed of the game. For Constant - Game Speed do not change when users clear the line,

To play the game demo a Unity Web player can be played here http://bit.ly/11pYNpJ

For those who do not have unity web player please download it here http://unity3d.com/webplayer/

Developer Story Background

This Tetris game is developed in 10 days, we are going to share the stories on how we build the game in 10 days. Originally, the game is an old experiment game software that was developed in 2011, however the earlier research student completed his master program and could not continue the experiment and the Game was picked up again in 2013 by another Student who would like to use the game for his experiment and thus began the story of rebuilding the game.

After looking at the old game code, and tried to run the run it unity, realized that the skin is missing and could not locate he old skin file as the game lies dormant for 2 years. Thus the search to find a collaborator to support the game development, We found Umair Ejaz (@habupain) who is very interested in supporting our development progress . and after chatting, we decided that the best way to build the game is to rebuild the game from scratch as the previous game , thus the story begins on how we build the Tetris Game in about 10 days.

Developer Story

Originally, the game is developed using Java Script using 3D,however in order to improve performance on multiple platform for future development including the low memory mobile phone app and to improve our lab code literacy, we decided to rewrite the whole logic from Javascript to C#. Thus began the whole process of development of the game .

The Game is developed using Scrum methods and using a simple task to do list app called asana, we manage to organize the tasks collaboratively through different country and timezone. Its a 10 day straight coding , testing, debugging and iterating and more testing. The game takes into consideration from alot of the aspect from from performance and memory management to design language. In addition, we conduct constant scrumming to discuss about current progress and plan our short term tasks.
